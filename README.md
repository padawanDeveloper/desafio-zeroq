## Como comenzar

-   Clonar repositorio

```bash
https://github.com/padawanDeveloper/desafio-zeroq.git
```

-   Instalar dependencias

```bash
npm install
```

-   Correr el proyecto

```bash
npm run dev
# or
yarn dev
# or
pnpm dev

bun dev
```
